# Toggle Theme Button

Toggle the theme between light and dark.

## Demo

<Canvas>
  <mrd-toggle-theme></mrd-toggle-theme>
</Canvas>

```html
<mrd-toggle-theme></mrd-toggle-theme>
```

## API

### Properties

| Property        | Attribute         | Type      | Default | Description                                                                                              |
| --------------- | ----------------- | --------- | ------- | -------------------------------------------------------------------------------------------------------- |
| `saveToStorage` | `save-to-storage` | `boolean` | `true`  | If true, the theme will be stored in local storage. If false, it will be stored in a cookie.             |
| `_theme`        | `theme`           | `string`  | `light` | The current theme. Can be either `light` or `dark`. This is used for the `data-theme` attribute on body. |

### Methods

| Method        | Type       | Description                                                                                                                                                 |
| ------------- | ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `toggleTheme` | `() => {}` | Toggles the theme between light and dark. Dispatches a `toggle-theme` event with the new theme as detail.                                                   |
| `initTheme`   | `() => {}` | Initializes the theme. If a theme is stored in local storage, it will be set as the theme. Otherwise, if the user prefers dark mode, dark mode will be set. |
| `store`       | `() => {}` | Stores the theme in local storage.                                                                                                                          |

### Events

| Event          | Description                                                                                  |
| -------------- | -------------------------------------------------------------------------------------------- |
| `toggle-theme` | Dispatched when the theme is toggled. The detail contains the new theme as `theme` property. |
