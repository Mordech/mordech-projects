# Chip

A pill-shaped label used to represent metadata, filters, or tags. Supports three
behavior variants: a static label, a clickable chip, or a chip with a remove button.

## Installation

```javascript
import '@mordech/web-components';
```

## Properties

| Property  | Type                                               | Default     | Description                                                                  |
| --------- | -------------------------------------------------- | ----------- | ---------------------------------------------------------------------------- |
| `color`   | `'primary' \| 'secondary' \| 'success' \| 'error'` | `'primary'` | Semantic color variant — maps to the tonal container/onContainer token pair. |
| `variant` | `'default' \| 'interactive' \| 'clearable'`        | `'default'` | Controls chip behavior (see Variants below).                                 |

## Variants

### Default

A static label. No interaction affordances.

<Canvas>
  <mrd-chip>Label</mrd-chip>
</Canvas>

```html
<mrd-chip>Label</mrd-chip>
```

### Interactive

Adds `role="button"`, `tabindex="0"`, keyboard activation (Enter / Space), and a
hover/active interaction layer. Use when the whole chip should be clickable.

<Canvas>
  <mrd-chip variant="interactive">Clickable</mrd-chip>
</Canvas>

```html
<mrd-chip variant="interactive">Clickable</mrd-chip>
```

### Clearable

Shows a remove (✕) button with its own hover/active interaction layer. Dispatches
`mrd-chip-clear` when clicked. Use for filter tags or dismissible labels.

<Canvas>
  <mrd-chip variant="clearable">Remove me</mrd-chip>
</Canvas>

```html
<mrd-chip variant="clearable">Remove me</mrd-chip>
```

## Events

| Event            | Detail | Description                                                                                                           |
| ---------------- | ------ | --------------------------------------------------------------------------------------------------------------------- |
| `mrd-chip-clear` | —      | Fired when the clear button is clicked (`variant="clearable"`). Does not bubble through the chip's own click handler. |

## Slots

| Name         | Description                                                 |
| ------------ | ----------------------------------------------------------- |
| default      | The chip label text.                                        |
| `icon-start` | An icon placed before the label text. Sized to 1rem × 1rem. |

## Parts

| Part         | Description                   |
| ------------ | ----------------------------- |
| `chip`       | The inner chip container div. |
| `label`      | The label `<span>`.           |
| `clear`      | The clear `<button>` element. |
| `icon-start` | The icon-start slot wrapper.  |

## Clearable — Implementation Example

Listen for `mrd-chip-clear` to remove chips from a list:

```html
<div id="chips">
  <mrd-chip variant="clearable">React</mrd-chip>
  <mrd-chip variant="clearable">TypeScript</mrd-chip>
  <mrd-chip variant="clearable">Lit</mrd-chip>
</div>

<script>
  document.getElementById('chips').addEventListener('mrd-chip-clear', (e) => {
    e.target.remove();
  });
</script>
```
