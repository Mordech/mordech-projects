# Typography

Typography is the base of all design systems.
It is the foundation of all visual communication. It is the way we communicate with our users.

## accessibility

When using typography,
you should always use the correct HTML element for the content you are trying to display.
For example, if you are trying to display a heading, you should use the `<Typography asElement="h1">` element.
If you are trying to display a paragraph, you should use the `<Typography asElement="p">` element. This will ensure that the content is accessible to all users.

## Tokens

> ⚠️ _Note:_ These tokens moved to the `@mordech/tokens` package. You can import them from there.
> Using the `@mordech/components` package for tokens is deprecated.

### Using typography tokens in your project

I have created a set of tokens for the typography.
You can use the typography tokens in your project by importing them from the `@mordech/tokens` package.

```jsx
import { fontSizes, fontWeights, fontFamilies } from '@mordech/tokens';

const Heading = styled.h1`
  font-size: ${fontSizes[4]};
  font-weight: ${fontWeights.bold};
  font-family: ${fontFamilies.sans};
`;
```

### Font Families

Font families are the typefaces used in your project.
You can use the font families in your project by importing them from the
`@mordech/tokens` package.

We use `Serif` and `Sans-serif`, and `Monospace` fonts in our project.

You should use the font family that is most appropriate for the content you are trying to display.
For example, if you are trying to display a heading, you should use the `Serif` font family.
If you are trying to display a paragraph, you should use the `Sans Serif` font family.

The Serif font is recommended for headings and titles `fontSizes[3]` and above.
The Sans Serif font is recommended for body copy and headings `fontSizes[2]` and below.

> If you are unsure which font family to use, you should ask your designer.

| fontFamilies | Value                       |
| ------------ | --------------------------- |
| sans         | 'DM Sans', sans-serif       |
| serif        | 'Eczar', serif              |
| code         | 'JetBrains Mono', monospace |
| inherit      | inherit                     |

#### Usage

```jsx
import { fontFamilies } from '@mordech/tokens';

const Heading = styled.h1`
  font-family: ${fontFamilies.sans};
`;
```

```jsx
import { fontFamilies } from '@mordech/tokens';

const Content = () => (
  <Typography asElement="p" fontFamily="serif">
    This is a paragraph
  </Typography>
);
```

### Font sizes

Font sizes are not bound to the `lineHeight` of the font family.
Also, font sizes are not bound to semantic elements.

| fontSizes | Value    | In px   |
| --------- | -------- | ------- |
| 0         | 0.875rem | 14px    |
| 1         | 1rem     | 16px    |
| 2         | 1.5rem   | 24px    |
| 3         | 2.25     | 36px    |
| 4         | 3rem     | 48px    |
| 5         | 3.375rem | 54px    |
| inherit   | inherit  | inherit |

import { fontSizes } from '@mordech/tokens';

const Heading = styled.h1`
